# Schema Validation Rules

## Global Rules
- Every city must have a 3-letter ID (e.g., BLR, BOM).
- Every entry must have an emoji representative.

## slang.md
- Keys: phrase (string), meaning (string), emoji (string).

## food_moods.md
- Keys: mood (string), dish (string), emoji (string).

## travel_rules.md
- Keys: mode (string), crowd (string), time_factor (float), health_karma (int), carbon_score (int).

## heritage.md
- Keys: fact (string), caption (string).
